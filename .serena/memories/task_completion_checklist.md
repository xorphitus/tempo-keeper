# Task Completion Checklist

When completing a task, run these commands in order:

1. **Format Code**
   ```bash
   npm run format
   ```

2. **Lint Code**
   ```bash
   npm run lint
   ```

3. **Run Tests**
   ```bash
   npm test
   ```

4. **Build (if applicable)**
   ```bash
   npm run build
   ```

All commands should pass without errors or warnings before considering a task complete.
