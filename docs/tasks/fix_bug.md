# バグ修正
この Web アプリケーションは以下の機能を有します。

1. 指定した BPM で拍の頭に音を鳴らすメトロノームです
2. Time signature で拍子を指定できます
3. ただし通常のメトロノームと異なり、Measures text input に数字を入れると、一定間隔で音が消えます
  - 入力は整数のみで、ここでは N とします
  - N 小節中、最初の1小節だけメトロノームの音が出て、残る N-1 小節は音をミュートします
  - N=1のとき、必ず全ての音が出ます
  - 例えばN=4のとき、現在M小節目だとすると M%4==1 の小節では音が出て、それ以外の小節ではミュートされます

現在このミュートが適切に機能しておらず、音が鳴って欲しいとこで鳴りません。
このバグを修正してください。
